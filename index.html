Conception & Matériaux

Conçu pour Durer

La structure du banc repose sur du pin de classe 3, commandé dans une scierie locale, adapté à un usage en extérieur. L'ensemble a été modélisé sous SolidWorks.

Les simulations mécaniques sous SolidWorks ont permis de valider la résistance structurelle à 300 kg, tandis que le boîtier électronique IP65 protège tous les composants sensibles des intempéries.

Structure en pin classe 3 — scierie locale
Compartiment batterie étanche IP65
Panneau solaire vissé sur cadre aluminium — inclinaison 37° (optimale pour la Somme, latitude 50°N)
Boîtier électronique ventilé passif
Prises USB-C et USB-A encastrées protégées
Honnêteté Technique

Limites du Prototype & Vision V2

Transparence sur les contraintes actuelles et les pistes d'amélioration.

Raspberry Pi 3B

Limite : Énergivore (~5W continu), surdimensionné pour la tâche
V2 : Pi Zero 2W (~0.5-1W), autonomie ×5

Batterie GEL 100Ah

Limite : ~500 cycles à 50% DOD (~3,3 ans). En pratique, DOD quotidien bien inférieur → durée de vie réelle 5-7 ans
V2 : LiFePO4 (3000+ cycles, -20°C à +60°C)

Aucune connexion réseau

Limite : Affichage local seulement, pas de monitoring à distance
V2 : Module 4G + dashboard web distant

Bilan hivernal

Limite : Production < consommation dans la Somme en janvier
V2 : Délestage automatique + veille profonde

Autonomie

Le contrôleur MPPT Victron SmartSolar 150|35 optimise en permanence le point de puissance maximale du panneau, offrant un gain de +30% de rendement par rapport à un régulateur PWM classique.

Avec une consommation moyenne d'environ 5W (de 5W en veille à 62W en pic — Raspberry Pi, écran, USB-C pleine charge), le système assure une disponibilité supérieure à 99% en Hauts-de-France. La puissance des ports USB est bridée logiciellement pour préserver l'autonomie. Le mode éco s'active sous 20% de batterie.

Fiche technique – Consommation énergétique

Configuration :

  • Raspberry Pi 3B+
  • Écran LCD 7" (activation par présence)
  • Capteur PIR
  • 2 × USB-C + 2 × USB-A
  • Convertisseur DC-DC

Usage :

  • 8 à 12 utilisateurs / jour
  • ~10 recharges quotidiennes
  • ~15 Wh utile / recharge (≈18 Wh avec pertes)
  • Jusqu'à 4 appareils simultanés

Consommation :

  • Système (24h) : 120 Wh/j
  • Écran : 20 Wh/j
  • USB : 150–180 Wh/j
Total retenu : ~300 Wh/jour

Caractéristiques électriques :

  • Puissance moyenne : ~12.5 W
  • Puissance max instantanée : ~60 W

Dimensionnement :

  • Valeur incluant usage réel + marge de sécurité
  • Pic possible : ~400 Wh/jour
